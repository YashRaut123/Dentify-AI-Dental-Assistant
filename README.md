# 🦷 Dentify - AI Dental Assistant

Dentify is an AI-powered dental assistant and appointment booking platform that helps users get instant voice-based dental guidance and schedule appointments easily.

---

## 🚀 Features

- 🎙️ AI Voice Assistant for dental queries  
- 📅 Book and manage appointments  
- 🧑‍⚕️ Admin dashboard to manage doctors  
- ⚡ Real-time updates and smooth UI  
- 🔐 Authentication system  

---

## 🛠️ Tech Stack

- Next.js  
- React  
- TypeScript  
- Tailwind CSS + shadcn/ui  
- Prisma  
- Vapi AI (Voice Assistant)
